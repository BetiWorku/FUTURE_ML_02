# FUTURE_ML_02
Support Ticket Classification (NLP)
CIN: FIT/FEB26/ML5800
   Project Overview
Automated classification of customer support tickets using Machine Learning and NLP . The model sorts tickets into categories like Technical, Billing, and Account.
   Key Results
Accuracy : 100% (1.00) on the test set.
Confusion Matrix : Achieved a perfect diagonal with zero misclassifications .
Dataset : Processed customer_support_tickets.csv(3.8 MB).

  Technical Workflow
Preprocessing : Handled missing values ​​and cleaned text data.
Feature Extraction : Used TF-IDF to convert text into numerical vectors.
Modeling : Trained a Random Forest Classifier (Ensemble Learning).
Evaluation : Generated a Confusion Matrix and Classification Report to verify performance.

 Repository Files
FUTURE_ML_02.ipynb: Complete Python implementation and visualizations.
customer_support_tickets.csv: The raw dataset used for training.
